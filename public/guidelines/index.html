Before uploading an image please read our guidelines.

  • Uploaded file must be an image (obviously)
  • Image must be properly tagged

    1. Please add at least one character name to image tags

    2. If all characters featured in image are unknown add "unknown" tag to image

    3. If any character featured in image is known "unknown" tag should not be added

    4. Tags that are not related to characters' names should not be added

  • Image must be SFW (Safe For Work)

    *Images featuring explicit content will not be tolerated and will be removed

  • If uploaded image breaks the rules it will be removed
